# DevOracle AI

AI-powered SaaS platform for bug detection, pull request review, security analysis, repository auditing, and senior-level code review using FastAPI, Gemini AI, React, TypeScript, and PostgreSQL.

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Shadcn UI

### Backend
- FastAPI
- Python 3.12+
- SQLAlchemy
- PostgreSQL (Supabase)

### AI
- Google Gemini API

### Authentication
- JWT Authentication
- GitHub OAuth

### Deployment
- Firebase Hosting
- Railway
- Supabase PostgreSQL
